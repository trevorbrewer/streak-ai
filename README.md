> Automatically pulls daily schedules, enriches hitter matchups,
> engineers advanced features, and delivers ranked picks via email every morning.

## Tech Stack

- **Python 3.11** — core language
- **Anthropic Claude** — AI hitter analysis and scoring
- **MLB Stats API** — free schedule, pitcher, and lineup data
- **pybaseball** — Statcast advanced metrics
- **SendGrid** — automated email delivery
- **GitHub Actions** — CI/CD and daily pipeline automation
- **GitHub Pages** — live dashboard
